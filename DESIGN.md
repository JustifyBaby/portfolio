1. プロジェクト概要

アプリ名： シフト管理アプリ

目的／背景：文化祭のシフト管理を効率化するため

対象ユーザー：生徒スタッフ

開発期間：2024年6月〜2024年9月、2025年8月～2025年9月

| 機能                   | 内容                                                | 優先度 | 備考                                             |
| ---------------------- | --------------------------------------------------- | ------ | ------------------------------------------------ |
| シフト入力             | 担当者ごとに希望時間を入力                          | 高     | スプレッドシート連携                             |
| シフト表示             | 1日ごと／スタッフ別にシフト表示                     | 高     | スプレッドシートに色分けして表示                 |
| 通知                   | シフトの時間の5分前にメールでリマインド             | 中     | メールアドレスを暗号化してスプレッドシートに保存 |
| バリデーションチェック | 勤務コマを3コマ以上                                 | 低     | スプシの関数で対応                               |
| パスコード保護         | パスコードを設け万が一URLが漏洩しても耐えられる設計 | 高     | 環境変数に保存                                   |
| 出席番号順ソート       | データの追加で常に出席番号順にソート                | 高     | -                                                |

3. UI／画面構成／できること

生と情報入力画面 (メアド、名前、出席番号)

シフト入力画面

シフト確認画面

スプレッドシートは管理者のみ閲覧可能で直接編集で柔軟に変更可能

スプレッドシートなのでPDF化してクラスラインに送信が可能

4. 技術スタック

フロントエンド：HTML / CSS / JavaScript

バックエンド：GAS Webアプリ

データベース：Google スプレッドシート

5. データ構造

スプレッドシートのシート構成

データフォーマット例もここに記載
